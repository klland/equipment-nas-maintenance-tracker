# 設備與 NAS 維護追蹤系統

這是一個以家用 NAS、路由器、交換器與工作站為情境的維護追蹤 Demo。作品目標是模擬設備工程師常見的異常紀錄、維修狀態追蹤與簡單統計分析。

## 功能

- 新增設備異常紀錄
- 紀錄設備名稱、異常類型、優先級、停機時間與處理方式
- 切換狀態：待處理、處理中、已完成
- 篩選不同狀態的維修紀錄
- 統計未完成案件、平均停機時間、完成件數與高優先級案件
- 顯示異常分類統計
- 使用 localStorage 保留資料

## 使用方式

直接用瀏覽器開啟 `index.html`。

## 面試說法

這個作品不是單純記錄資料，而是把設備異常轉成可以追蹤與分析的資訊。透過記錄停機時間、異常類型、處理方式與狀態，可以觀察哪些設備或問題類型容易重複發生，作為後續保養與改善的依據。
